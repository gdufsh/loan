# loan

贷款还款计划计算器，支持等额本金、等额本息，以及香港 H/P 利率联动场景下的浮动利率等额本息。

## 安装

需要 Python 3.10+。

```bash
pip install -e ".[dev]"
```

## 使用

### 等额本息

```
loan -p 1000000 -r 3.25 -m 360
```

### 等额本金

```
loan -p 1000000 -r 3.25 -m 360 -M equal-principal
```

### 对比两种还款方式

```
loan -p 1000000 -r 3.25 -m 360 --compare
```

加 `--compare-detail` / `-d` 同时输出逐期明细。

### 浮动利率等额本息

`-r` 为封顶利率，`--rate-changes` 定义利率低于封顶值的区间，格式为 `x:y:rc`（第 x 至 y 期使用年利率 rc%），区间外自动回退至封顶利率。

```
loan -p 1000000 -r 3.25 -m 360 --rate-changes 1:24:2.5
```

多段区间用逗号分隔，区间不可重叠，rc 不可超过 -r：

```
loan -p 1000000 -r 3.25 -m 360 --rate-changes 1:24:2.5,25:36:3.0
```

### 纯文本输出

加 `--no-color` / `-n` 禁用 rich 彩色渲染，适合重定向到文件或管道：

```
loan -p 1000000 -r 3.25 -m 360 -n > schedule.txt
```

## 参数说明

| 参数 | 简写 | 说明 |
|---|---|---|
| `--principal` | `-p` | 贷款本金（元），如 `1000000` |
| `--annual-rate` | `-r` | 年利率（百分比），如 `3.25` 表示 3.25%；浮动模式下为封顶利率 |
| `--months` | `-m` | 还款期数（月），如 `360` |
| `--method` | `-M` | 还款方式：`equal-installment`（默认）、`equal-principal`；启用 `--compare` 时忽略 |
| `--compare` | `-c` | 对比等额本金与等额本息 |
| `--compare-detail` | `-d` | 对比模式下附加逐期明细（需配合 `--compare`） |
| `--rate-changes` | | 浮动利率区间，格式 `x:y:rc[,x:y:rc,...]` |
| `--no-color` | `-n` | 禁用彩色输出 |

`--compare` 与 `--rate-changes` 不可同时使用。

## 算法说明

- **等额本金**：每月还款本金固定，利息随剩余本金递减。
- **等额本息**：按 PMT 公式一次性计算固定月供，末期本金取剩余余额以消除舍入误差。
- **浮动利率等额本息**：每期按当期利率与剩余期数重新计算 PMT，期数固定不延长，末期同样取剩余余额归零。所有金额使用 `decimal.Decimal` 精确计算，舍入至分（`ROUND_HALF_UP`）。
